# 👨‍🍳🍳 Flutter Recipe Creator

A modern Flutter recipe application built with **Flutter** and **Dart** that allows users to browse and explore delicious recipes through a clean, responsive, and intuitive interface. The project follows a modular architecture with reusable widgets and REST API integration.

---

## ✨ Features

- 🍽️ Browse a variety of recipes
- 🔍 Search recipes by name
- 📖 View detailed recipe information
- 🥗 Display ingredients and cooking instructions
- 📱 Responsive and user-friendly UI
- 🧩 Reusable widgets
- 🌐 REST API integration
- 🚀 Smooth navigation

---

## 🛠️ Tech Stack

- 💙 Flutter
- 🎯 Dart
- 🌐 REST API
- 📦 HTTP Package
- 🎨 Material Design

---

## 📂 Project Structure

```
lib/
│
├── models/
├── pages/
├── services/
├── generated_routes.dart
└── main.dart
```

### Folder Description

- **models/** – Data models for recipes and API responses.
- **pages/** – Application screens and UI.
- **services/** – API calls and business logic.
- **generated_routes.dart** – Centralized navigation routes.
- **main.dart** – Application entry point.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-recipe-creator.git
```

### Navigate to the project

```bash
cd flutter-recipe-creator
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
